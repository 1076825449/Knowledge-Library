# 知识库使用指南

## 快速开始

### 搜索内容
在顶部搜索框输入关键词即可全文搜索，支持：
- 问题标题、简述、详细解答
- 政策文号（如"财政部 税务总局 公告2019年"）
- 标签名称

**示例关键词**：发票、红字发票、注销、清税、非正常户、小规模纳税人

### 筛选内容
列表页支持多维度筛选：
- **阶段**：设立期 / 经营期 / 注销期
- **模块**：税务登记 / 发票管理 / 申报纳税 / 社保公积金
- **问题类型**：操作类 / 政策类 / 概念类
- **高频问题**：勾选"只看高频"快速找到最常见问题
- **新手必看**：勾选"新手必看"找到入门级内容

### 查看详情
点击任意条目进入详情页，包含：
- 一句话结论（快速了解）
- 详细解答（含适用条件、例外边界、实操步骤、风险提示）
- 政策依据列表（可点击查看原文）
- 关联问题推荐
- 状态标签（正常/草稿/待审核/已退回）

---

## 政策依据使用说明

每条内容可能关联多条政策依据：
- **来源**：国家税务总局、财政部、各省税务局
- **状态**：已核实 / 来源待核实 / 待更新 / 需人工复核
- **查看原文**：点击政策名称跳转来源网站

**注意**：标注"待更新"的政策可能在2026年新法实施后失效，请以最新法规为准。

## 常见问题

**Q：为什么搜不到刚更新的内容？**
A：新发布的内容可能有延迟。若问题状态为"待审核"则只对编辑/审核人员可见。

**Q：内容有误怎么办？**
A：详情页右上角有"纠错反馈"链接，可提交修改建议。

**Q：什么是"高频问题"？**
A：标记为高频的问题是最常见的税务咨询，点击筛选"只看高频"可快速找到。
